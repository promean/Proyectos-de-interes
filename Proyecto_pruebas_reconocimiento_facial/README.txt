# Proyecto de reconocimiento facial

En este proyecto mi objetivo es desarrollar un programa con el objetivo de que se pueda reconocer las expresiones faciales de una persona. Todavía está en fase de desarrollo, 
hay que decidir que tecnologías se van a implementar y cuáles son los campos que se van a medir. Por ejemplo:

- Calidad cámara: Éste programa está pensado para que funcione en todos los dispositivos, pero hay algunas cámaras, sobretodo las que tienen calidad de 400p, que fallan mucho a la hora de
  detectar expresiones, por lo que también va a haber un script con el objetivo de medir la intensidad de las expresiones que pueda medir la cámara para cada usuario con el objetivo de 
  tener una experiencia mas personalizada para cada uno.
